# My Barisal (মাই বরিশাল) 🏛️

**My Barisal** is a comprehensive service-oriented mobile application designed to empower the citizens of Barisal. Built with **React Native** and **Expo**, the app acts as a digital hub for essential local information, emergency services, and citizen resources.

---

## 🚀 Features

### 🏥 Hospital Directory
- Access a complete list of hospitals, clinics, and diagnostic centers in Barisal.
- Direct calling feature for emergency contact.
- Integrated API support for real-time data updates.

### 💊 Pharmacy & Healthcare
- Locate nearby pharmacies and essential medicine shops.
- Categorized services for easy navigation.

### 🎓 Citizen & Education Services (নাগরিক ও শিক্ষা সেবা)
- Information regarding local educational institutions.
- Access to various government and municipal services for citizens.

### ⚡ Performance & UI/UX
- **Modern UI:** Clean and intuitive interface designed with custom iconography.
- **Skeleton Loaders:** Enhanced user experience during data fetching.
- **Optimized Routing:** Powered by **Expo Router** for seamless screen transitions.
- **Bengali Language Support:** Fully localized interface for better accessibility.

---

## 🛠️ Tech Stack

- **Framework:** [React Native](https://reactnative.dev/) / [Expo](https://expo.dev/)
- **Language:** [TypeScript](https://www.typescriptlang.org/)
- **Navigation:** [Expo Router (File-based)](https://docs.expo.dev/routing/introduction/)
- **State Management:** React Hooks (useState, useEffect)
- **Styling:** React Native StyleSheet / Custom Design System
- **API Handling:** Axios / Fetch API
